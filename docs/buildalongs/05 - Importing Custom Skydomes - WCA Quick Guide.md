# Importing Custom Skydomes in Meta Horizon Worlds Desktop Editor

###  Related Links:
[Developer Documentation: Importing Custom Skydomes](https://developers.meta.com/horizon-worlds/learn/documentation/desktop-editor/preparing-skydome-maps-for-horizon-worlds-ingestion)
## When to use:
*Use this when you are ready to import a custom skydome to use within your world*
To upload a custom skydome in edit mode: 
1. Open the **Asset Library** and select **My Assets**\
![][image1]
2. **Select the folder** where you want to save your asset\
![][image2]
3. Click **Add New** and select **Skydome**\
![][image3]    
4. **Note:** **Custom skydomes have 4 components**. **The** **Display Map is the only component that is NOT optional**\
![][image4]
5. For each map you wish to upload, select **Choose files on your device**, then **select the mapâ€™s file** in the file browser popup window, then **press Open**\
![][image5]
6. When all desired maps have been added, **press Next**\
![][image6]   
7. **Provide a name and description** for the skydome asset, then press **Done**\
![][image7]
8. **Click and drag the uploaded skydome into your scene**\
![][image8]

### You now have a custom skydome ready to use in your world\!
